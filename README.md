# 🎬 Netflix Style Movie Recommendation System

A content-based Movie Recommendation System built using Python, Pandas, Scikit-Learn, Streamlit, and TMDB API.

## 🚀 Live Demo

https://movie-recommender-4me5vtbpxcaf4jtqcm8tu3.streamlit.app/

---

## 📌 Features

* Movie recommendation based on content similarity
* Movie posters from TMDB
* Movie ratings and release dates
* Movie overview/details
* Watch official movie trailers
* Similarity score (%) for recommended movies
* Responsive Streamlit UI
* Deployed on Streamlit Cloud

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Streamlit
* TMDB API
* Pickle

## ⚙️ Installation

Clone the repository:

git clone https://github.com/mohd-sarfraz/movie-recommender.git

Move into the project folder:

cd movie-recommender

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py

---

## 🔑 Environment Variables

Create Streamlit Secrets and add:

TMDB_API_KEY = "YOUR_TMDB_API_KEY"

## 🎯 Recommendation Method

This project uses Content-Based Filtering.

Movies are recommended based on similarity between movie features using cosine similarity.
